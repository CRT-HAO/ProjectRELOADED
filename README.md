# ProjectRELOADED
 本地化專案管理器
 ## 下載 Downloads
 [Releases](https://github.com/CRT-HAO/ProjectRELOADED/releases)
 ## 測試&編譯 Debug & Build
 安裝模組 Install modules
 ```
 npm install
 ```
 啟動測試 Start debug
 ```
 npm start
 ```
 編譯 Build
 ```
 node build.js
 ```
